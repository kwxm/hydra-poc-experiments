Description

Balance and sign the given partial transaction.

XXX(SN): This function does retry when no payment UTXO was found and thus might block. This is necessary in some situations when the wallet has not yet "seen" the change output although the direct chain client observed a transaction already. This is a smell and we should try to avoid having this race condition in the first place.
